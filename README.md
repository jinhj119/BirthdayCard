# Birthday Card

A simple desktop birthday card application built with Java Swing. The application displays a birthday message when the user clicks the **Open Card** button and includes custom images for a cake and balloons.

## Features

* Graphical user interface built with Java Swing
* Button click event handling using `ActionListener`
* Custom birthday message
* Cake and balloon images
* Custom background color
* Packaged as a Windows desktop application using `jpackage`

## Technologies Used

* Java 26
* Java Swing
* Eclipse IDE

## Project Structure

```text
src/
├── Main.java
├── BirthdayWindow.java
├── BirthdayListener.java
└── images/
    ├── cake.png
    └── balloon.png
```

## How to Run

1. Clone this repository.
2. Open the project in Eclipse or another Java IDE.
3. Run `Main.java`.
